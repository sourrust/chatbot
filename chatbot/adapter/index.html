Structs

CliAdapter

The CliAdapter reads lines from stdin and dispatches them as IncomingMessages to the chatbot. Replies are printed to stdout. There is currently no extra configuration available to the CliAdapter.
IrcAdapter

Connect your bot to IRC with the IrcAdapter
SlackAdapter

SlackAdapter sends and receives messages from the Slack chat service. Until actualy configuration is added, the slack token should be placed in the environment variable SLACK_BOT_TOKEN

Traits

ChatAdapter

Chatbot is extensible in both message sources and command handling. To add a new message source, create a type that implements the ChatAdapter trait.
