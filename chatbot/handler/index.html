MessageHandler

Implementing a MessageHandler enables responding to IncomingMessages. There are currently very few requirements to creating a handler. The handle function receives IncomingMessages. The implementer is responsible for testing whether it's interested in replying. Optionally call reply on the IncomingMessage to send a response.
